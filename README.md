# 涉诈网址历史访客风险等级分级模型
这是一个每天定时对访客进行风险等级分级的服务。
## 项目结构

```
app/
├── logs/              # 日志
├── core/              # 日志处理器
├── model/             # 模型文件
├── output/            # 分级结果
├── services/          # 处理数据
├── config.py          # 基本配置信息
├── main.py            # 应用入口
├── run.sh             # 启动脚本
├── test.py            # 测试入口
└── run_test.sh        # 测试启动脚本
```
